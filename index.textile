h2. Introduction

What the "vim":http://www.vim.org/ world _really_ needed is another colourscheme, so here it is.

!default.png!

If you prefer darker backgrounds, use the following:

{% highlight vim %}
let g:jnrowe_dark = 1
colorscheme jnrowe
{% endhighlight %}

With a @g:jnrowe_dark@ set it will look more like this:

!dark.png!
